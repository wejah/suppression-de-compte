Suppression de compte Coursak

Comment supprimer votre compte

Pour supprimer votre compte Coursak, suivez ces étapes simples :

  1. Ouvrez l'application Coursak
  2. Connectez-vous à votre compte
  3. Ouvrez le menu latéral (en appuyant sur l'icône de menu en haut à gauche)
  4. Faites défiler jusqu'à la fin du menu
  5. Appuyez sur "Supprimer le compte"
  6. Confirmez votre choix dans la boîte de dialogue

Données supprimées

Lors de la suppression de votre compte, les données suivantes seront définitivement supprimées :

  • Vos informations de profil (nom, numéro de téléphone)
  • Votre compte d'authentification

Données conservées

Certaines données peuvent être conservées pour des raisons légales ou de sécurité :

  • Les données nécessaires pour prévenir la fraude ou assurer la sécurité de nos services
Note importante : La suppression de votre compte est irréversible. Assurez-vous de sauvegarder toutes les informations importantes avant de procéder à la suppression.
